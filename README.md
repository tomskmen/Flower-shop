# 🌸 Flower Shop - Лендинг цветочного магазина

Современный одностраничный сайт (лендинг) для продажи цветов с адаптивным дизайном и интерактивными элементами.

## 📋 Описание проекта

Это полнофункциональный лендинг для цветочного магазина, включающий:
- Современный дизайн с красивыми шрифтами и анимациями
- Адаптивная верстка для всех устройств
- Интерактивные элементы и формы
- Каталог букетов с возможностью заказа
- Секции: главная, преимущества, каталог, о нас, доставка, отзывы, контакты

## 🚀 Быстрый старт

1. **Клонируйте или скачайте проект**
2. **Загрузите изображения:**
   - Откройте файл [`download-images.html`](download-images.html:1) в браузере

3. **Откройте [`index.html`](index.html:1)** в браузере

Вот и всё! Сайт готов к работе с красивыми фотографиями цветов!

## 🎨 Особенности дизайна

### Современный хедер:
- Полупрозрачный фон с эффектом размытия (backdrop-filter)
- Градиентная полоса сверху
- Анимированный логотип с эффектом плавания
- Стильное меню с декоративными элементами
- Плавные переходы и hover-эффекты

### Hero-секция:
- Полноэкранный фон с изображением букета
- Градиентный оверлей для читаемости текста
- Крупные заголовки с красивыми шрифтами
- Анимация появления элементов
- Волновой SVG-элемент внизу

### Шрифты:
- **Cormorant Garamond** - для заголовков (элегантный serif)
- **Montserrat** - для основного текста (современный sans-serif)
- **Dancing Script** - для логотипа (рукописный)

### Цветовая палитра:
- Основной: `#e91e63` (розовый)
- Вторичный: `#c2185b` (темно-розовый)
- Акцент: `#f06292` (светло-розовый)
- Золотой: `#d4af37` (для декоративных элементов)
- Темный: `#1a1a2e` (для текста)

## 📝 Настройка контента

### Изменение контактной информации:

Откройте `index.html` и найдите секцию `<section id="contacts">`:

```html
<a href="tel:+79001234567">+7 (900) 123-45-67</a>
<a href="mailto:info@flowershop.ru">info@flowershop.ru</a>
<p>г. Москва, ул. Цветочная, д. 15</p>
```

### Изменение цен и названий букетов:

Найдите секцию `<section id="catalog">` и отредактируйте:

```html
<h3 class="product-title">Название букета</h3>
<p class="product-description">Описание букета</p>
<span class="product-price">2 500 ₽</span>
```

### Изменение цветовой схемы:

Откройте `styles.css` и измените переменные в `:root`:

```css
:root {
    --primary-color: #e91e63;    /* Основной цвет */
    --secondary-color: #c2185b;  /* Вторичный цвет */
    --accent-color: #f06292;     /* Акцентный цвет */
    --gold: #d4af37;             /* Золотой */
}
```

## 🌐 Развертывание

### GitHub Pages:
1. Загрузите проект в GitHub репозиторий
2. Перейдите в Settings → Pages
3. Выберите ветку main и папку root
4. Сохраните - сайт будет доступен по адресу `username.github.io/repository`

### Netlify:
1. Перетащите папку проекта на [netlify.com/drop](https://app.netlify.com/drop)
2. Сайт автоматически развернется и получит уникальный URL

### Vercel:
1. Установите Vercel CLI: `npm i -g vercel`
2. В папке проекта выполните: `vercel`
3. Следуйте инструкциям
