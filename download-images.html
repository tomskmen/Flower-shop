🌸 Загрузка изображений для Flower Shop

Ниже представлены прямые ссылки на качественные бесплатные изображения с Unsplash. Нажмите на кнопку "Скачать", чтобы загрузить изображение, затем сохраните его в папку images/ с соответствующим названием.

📋 Инструкция по загрузке:

  1. Нажмите на кнопку "Скачать" под нужным изображением
  2. Изображение откроется в новой вкладке
  3. Нажмите правой кнопкой мыши → "Сохранить изображение как..."
  4. Сохраните в папку images/ с указанным названием файла
  5. Повторите для всех изображений
