# Папка для изображений

## Инструкция по добавлению фотографий

В эту папку нужно добавить следующие изображения:

### Обязательные изображения:

1. **hero-bg.jpg** - Фоновое изображение для главного экрана (рекомендуемый размер: 1920x1080px)
   - Красивый букет цветов для фона hero-секции

2. **about-image.jpg** - Изображение для секции "О нас" (рекомендуемый размер: 800x600px)
   - Фото флориста за работой или красивая композиция

### Изображения для каталога букетов:

3. **product-1.jpg** - Букет "Романтика" (красные розы)
4. **product-2.jpg** - Букет "Солнечное настроение" (подсолнухи)
5. **product-3.jpg** - Букет "Весенняя нежность" (тюльпаны)
6. **product-4.jpg** - Букет "Экзотика" (орхидеи)
7. **product-5.jpg** - Букет "Полевые цветы"
8. **product-6.jpg** - Букет "Персиковый рассвет"

Рекомендуемый размер для изображений букетов: 600x600px

## Где найти изображения:

- **Unsplash** (https://unsplash.com) - бесплатные качественные фото
- **Pexels** (https://pexels.com) - бесплатные стоковые фото
- **Pixabay** (https://pixabay.com) - бесплатные изображения

### Поисковые запросы:
- "flower bouquet"
- "roses bouquet"
- "sunflowers"
- "tulips bouquet"
- "orchids"
- "wildflowers"
- "florist working"

## Примечание:

После добавления изображений в эту папку, сайт автоматически будет использовать их вместо градиентных заглушек.
